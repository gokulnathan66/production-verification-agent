# Knowledge Base Architecture

## Overview
The Knowledge Base (KB) serves as the shared memory and context store for all agents, enabling intelligent decision-making based on historical data, policies, and real-time state.

## Recommended Technology by Stage

### Phase 1 (Local): PostgreSQL + Redis
**Why PostgreSQL:**
- JSONB support for flexible schema
- Full-text search capabilities
- Production-ready from day one
- Easy migration to cloud (RDS)
- Run locally via Docker

**Why Redis:**
- Real-time state caching
- Fast agent status updates
- Pub/sub for notifications
- Session management

### Phase 2 (Cloud): PostgreSQL (RDS) + Redis (ElastiCache)
- Managed services on AWS
- Auto-scaling and backups
- Multi-AZ deployment

### Phase 3 (Production): PostgreSQL + Redis + Vector DB
**Add Vector DB (pgvector extension):**
- Semantic search for code patterns
- AI/ML embedding storage
- Similar vulnerability detection

CREATE TABLE projects (

    name TEXT NOT NULL,

CREATE TABLE project_languages (
    project_id TEXT REFERENCES projects(id),
    language TEXT NOT NULL,

    PRIMARY KEY (project_id, language)

-- project_dependencies table
CREATE TABLE project_dependencies (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    project_id TEXT REFERENCES projects(id),
    name TEXT NOT NULL,
    version TEXT,
    language TEXT,
    is_dev BOOLEAN DEFAULT FALSE
);

-- project_architecture table
CREATE TABLE project_architecture (
    project_id TEXT PRIMARY KEY REFERENCES projects(id),
    architecture_type TEXT,    -- 'monolith', 'microservice', 'serverless'

    detected_patterns JSON

### PostgreSQL Connection
```python
# kb/database.py
import asyncpg
import redis.asyncio as redis
from contextlib import asynccontextmanager

class KnowledgeBase:
    def __init__(self, pg_dsn: str = "postgresql://pva:pva_local@localhost:5432/pva_kb",
                 redis_url: str = "redis://localhost:6379"):
        self.pg_dsn = pg_dsn
        self.redis_url = redis_url
        self.pool = None
        self.redis = None
    
    async def connect(self):
        self.pool = await asyncpg.create_pool(self.pg_dsn, min_size=2, max_size=10)
        self.redis = await redis.from_url(self.redis_url)
    
    async def close(self):
        await self.pool.close()
        await self.redis.close()
    
    @asynccontextmanager
    async def connection(self):
        async with self.pool.acquire() as conn:
            yield conn
    
    def get_retriever(self) -> 'KBRetriever':
        return KBRetriever(self.pool, self.redis)
    
    def get_saver(self) -> 'KBSaver':
        return KBSaver(self.pool, self.redis)
```

### Retriever Implementation
```python
# kb/retriever.py
class KBRetriever:
    def __init__(self, pool: asyncpg.Pool, redis: redis.Redis):
        self.pool = pool
        self.redis = redis
    
    async def get_project(self, project_id: str) -> dict:
        # Check Redis cache first
        cached = await self.redis.get(f"project:{project_id}")
        if cached:
            return json.loads(cached)
        
        async with self.pool.acquire() as conn:
            row = await conn.fetchrow(
                "SELECT * FROM projects WHERE id = $1", project_id
            )
            if row:
                data = dict(row)
                await self.redis.setex(f"project:{project_id}", 300, json.dumps(data))
                return data
        return None
    
    async def get_security_policies(self, languages: list) -> list:
        async with self.pool.acquire() as conn:
            return await conn.fetch("""
                SELECT * FROM security_policies 
                WHERE enabled = true AND languages ?| $1
            """, languages)
    
    async def get_historical_vulns(self, pattern: str, limit: int = 10) -> list:
        async with self.pool.acquire() as conn:
            return await conn.fetch("""
                SELECT * FROM vulnerabilities 
                WHERE description ILIKE $1 
                ORDER BY detected_at DESC LIMIT $2
            """, f"%{pattern}%", limit)
    
    async def get_agent_status(self, session_id: str) -> dict:
        # Real-time from Redis
        status = await self.redis.hgetall(f"session:{session_id}:agents")
        return {k.decode(): json.loads(v) for k, v in status.items()}
```

### Saver Implementation
```python
# kb/saver.py
class KBSaver:
    def __init__(self, pool: asyncpg.Pool, redis: redis.Redis):
        self.pool = pool
        self.redis = redis
    
    async def save_project(self, project: dict) -> str:
        async with self.pool.acquire() as conn:
            project_id = await conn.fetchval("""
                INSERT INTO projects (id, name, source_type, source_url)
                VALUES ($1, $2, $3, $4) RETURNING id
            """, project['id'], project['name'], 
                project.get('source_type'), project.get('source_url'))
        return project_id
    
    async def save_finding(self, session_id: str, finding: dict) -> int:
        async with self.pool.acquire() as conn:
            finding_id = await conn.fetchval("""
                INSERT INTO current_findings 
                (session_id, agent_id, finding_type, severity, details, blocking)
                VALUES ($1, $2, $3, $4, $5, $6) RETURNING id
            """, session_id, finding['agent_id'], finding['finding_type'],
                finding['severity'], json.dumps(finding['details']), finding.get('blocking', False))
        
        # Update Redis for real-time access
        await self.redis.lpush(f"session:{session_id}:findings", json.dumps(finding))
        return finding_id
    
    async def update_agent_status(self, session_id: str, agent_id: str, status: dict):
        # Real-time in Redis
        await self.redis.hset(
            f"session:{session_id}:agents",
            agent_id,
            json.dumps(status)
        )
        # Persist to Postgres
        async with self.pool.acquire() as conn:
            await conn.execute("""
                INSERT INTO agent_status (session_id, agent_id, status, progress)
                VALUES ($1, $2, $3, $4)
                ON CONFLICT (session_id, agent_id) 
                DO UPDATE SET status = $3, progress = $4
            """, session_id, agent_id, status['status'], status.get('progress', 0))
    
    async def save_vulnerability(self, vuln: dict) -> int:
        async with self.pool.acquire() as conn:
            return await conn.fetchval("""
                INSERT INTO vulnerabilities 
                (project_id, cve_id, severity, category, file_path, line_number, description, remediation)
                VALUES ($1, $2, $3, $4, $5, $6, $7, $8) RETURNING id
            """, vuln['project_id'], vuln.get('cve_id'), vuln['severity'],
                vuln['category'], vuln['file_path'], vuln.get('line_number'),
                vuln['description'], vuln.get('remediation'))
```

## Redis Usage Pattern

| Key Pattern | Purpose | TTL |
|-------------|---------|-----|
| `project:{id}` | Cached project metadata | 5 min |
| `session:{id}:agents` | Real-time agent status | Session |
| `session:{id}:findings` | Live findings list | Session |
| `policies:cache` | Security policies cache | 1 hour |

## Migration Path

### Phase 1 → Phase 2
1. Create RDS PostgreSQL instance
2. Create ElastiCache Redis cluster
3. Update connection strings in config
4. Run `pg_dump` / `pg_restore` for data migration

### Phase 2 → Phase 3
1. Add pgvector extension to PostgreSQL
2. Create embedding tables for code patterns
3. Integrate vector search in retriever
